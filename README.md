# Automação de Tarefas com Python

Este projeto foi desenvolvido com o objetivo de automatizar tarefas repetitivas no computador utilizando Python.  
A automação realiza interações com o mouse e teclado, permitindo executar processos de forma mais rápida e eficiente.

## 📌 Tecnologias Utilizadas

- Python
- PyAutoGUI
- Pandas
- Time

## ⚙️ Funcionalidades

- Automação de cliques e movimentação do mouse
- Execução automática de tarefas repetitivas
- Leitura de dados a partir de planilhas
- Preenchimento automático de informações

## 📂 Estrutura do Projeto

main.py → Script principal da automação  
produtos.csv → Arquivo de dados utilizado na automação  

## ▶️ Como Executar o Projeto

1. Instale o Python em seu computador

2. Instale as bibliotecas necessárias:

pip install pyautogui pandas time

3. Execute o script:

python main.py

## 💡 Objetivo do Projeto

Este projeto foi desenvolvido como prática de programação e automação de processos utilizando Python, com foco em aumentar produtividade e reduzir tarefas manuais.
